# GenoDiplo Note

The original project guide suggested using the GenoDiplo pipeline with Snakemake.

For this project, I completed the genome assembly using an equivalent manual workflow with:
- FastQC
- Flye
- QUAST

This approach was chosen to ensure that the project could be completed successfully within the available time while still covering the essential genome assembly and evaluation steps.

The GenoDiplo repository was included for reproducibility and alignment with the suggested workflow.

# GenoDiplo Note

The original project guide suggested using the GenoDiplo pipeline with Snakemake.

For this project, I completed the genome assembly using an equivalent manual workflow with:
- FastQC
- Flye
- QUAST

A configuration-style record was also prepared in `config_used.yaml` to document the selected reads, estimated genome size, and workflow parameters.

This manual approach was chosen to ensure successful completion of the assembly and evaluation steps within the available time.

The GenoDiplo repository was included in the project directory to align the project structure with the original guide.
